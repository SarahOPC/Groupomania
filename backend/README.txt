Application servant de réseau social d'entreprise pour les salariés de Groupomania.
Cela leur permet d'interagir entre collègues de manière plus informelle afin d'améliorer l'ambiance de travail.
Le projet fait appel pour le frontend a un framework Vue.
Le backend est codé grâce à node.js et express.
La base de données est une base de données Sql avec comme SGBDR mysql.


Pour commencer :
npm install

Pour lancer le server backend :
npm start
Pour lancer le serveur frontend :
npm run serve


--------------------------------------------------------------------------------------------------------------

Application that is a corporate social network for Groupomania employees.
This allows them to interact with colleagues in a more informal way to improve the working atmosphere.
The project uses a Vue framework for the frontend.
The backend is coded using node.js and express.
The database is a Sql database with as RDBMS mysql.

To begin :
npm install

To run backend server :
npm start
To run frontend server :
npm run serve
